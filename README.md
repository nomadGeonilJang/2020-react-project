# 개발환경 셋팅하기
1. webpack-dev-server 설치하기
```bash
# npm i -D webpack webpack-cli webpack-dev-server
```
2. package.json 에서 webpack-dev-server 셋팅하기
```json
{
  "scripts": {
    "start": "webpack-dev-server"
  }
}
```
